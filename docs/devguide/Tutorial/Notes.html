<D></D>

M(x,y) : set point. Ex: <D>[M(0, 0)]</D>

L(x,y) : draw a line from a point to x,y. Ex: <D>[M(0, 0), L(10, 10)]</D>

z: draw a line from last point to first point.

C(x1, y1, x2, y2, x, y): draw a curve with absoluted end point x,y (see more C in SVG)

c(x1, y1, x2, y2, x, y): draw a curve with relative end point x,y (see more c in SVG)

S(x2, y2, x, y): draw a circle with relative end point x,y (see more S in SVG)
