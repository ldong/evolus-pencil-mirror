6. Drawing image


A Pencil shape sometimes have to contain svg images. This tutorial will show how to put an image to a shape.


Assume that having a hand image


An image will be parsed to base64 binary and embebed into a Pencil shape.There are many tools to parse images to base64 binary. Pencil also supports an image parser. Go Tools â†’ Developer Tools â†’
