Collection Properties

Shapes in a collection tend to have the same style such as same font, color, stroke style... To be convenient to stencil authors in this situation, Pencil supports grouping all shapes common style to collection style. Then if collection style is changed, all shapes refer to collection style will be changed accordingly, too.

Have a look at this sample. Collection properties are defined and then used in shape properties. 


  
    
      Arial|normal|normal|13px
      #000000ff
    
  
  
    
      
        Hello World
        $$defaultTextColor
        $$defaultTextFont
      
    

    
      ...
    

    
      
    
  

]]>

Text content inside the <Property> tag is a shape is the literal presentation of the initial value for that property. In this example you can notice that the content inside that tag is a <E></E> instead. This is the notation to imply that the initial value should be obtained by evaluation the 'expression' inside the <E> tag. To reference a specific collection property in side this expression, the $$ syntax is used.
